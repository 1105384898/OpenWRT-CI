# OpenWRT-CI
云编译OpenWRT固件

LEDE源码：
https://github.com/coolsnowwolf/lede

IMMORTALWRT源码：
https://github.com/immortalwrt/immortalwrt

# 固件简要说明：

LWRT——LEDE源码

IWRT——IMMORTALWRT源码

Rockchip——RK系列，包含R4S、R5C、R66S、R68S

X64——X64系列，包含X64、X86

X86——X86系列，仅包含X86

# 目录简要说明：

Depends.txt——环境依赖列表

Scripts——自定义脚本

Config——自定义配置

workflows——自定义CI配置
